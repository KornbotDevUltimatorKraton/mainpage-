Roboreactor is a platform that enables you to build robotic and automation systems more rapidly and with more capabilities. It enables you to manage general-purpose mobile computers in order to swiftly construct robotics and automation systems while decreasing task complexity in only three steps.

Create a project

Build the hardware for your robot and automation system. Create a project token and configure your device specifications by logging in and visiting the project page.Before you begin, please download the middleware for the single-board computer you are using in your robot. You can find the download link on the download page or at the top of this page. Additionally, click on 'Introducing Genflow mini' to redirect to the GitHub page, where you can find detailed installation instructions. Please read the instructions before proceeding with the installation.

Generate code

Download and flash the genflow image to your single board computer. You can now produce code for your robot single board computer extremely quickly with the Roboreactor node generator without having to write code or understand low level algorithm.

Create URDF or Upload URDF file

Using our urdf editor, you can upload and construct your CAD model, as well as add motion joints, to design the motion system.You can test the uploaded STL 3D model by clicking on the selection box to open it, and then experiment with dragging and assembling the components. You can now use our node generator to program and visualize it in real time with our motion control.

Controlling and visualizing

with our motion control you can remotely connect with your robot utilizing our rapid IoT system, allowing you to operate your robot's real-time from anywhere. It enables you to run the robot in development mode with no gaps in between.

Navigation system and visualizing

Real-time point cloud mapping, created by your robot's sensors including Lidar, Radar, Sonar, WiFi localization, Beacons, and cameras, is available for indoor and outdoor navigation systems through our navigation visualizer. Track your robot's movements and coordinate multiple robots to create a swarm.
